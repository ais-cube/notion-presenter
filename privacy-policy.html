Privacy Policy

Notion Presenter Chrome Extension — Last updated: March 15, 2026

Overview

Notion Presenter is a browser extension that transforms published Notion pages into presentation slides. It operates entirely within your browser and does not collect, store, or transmit any user data.

Data Collection

This extension does not collect any personal information, usage data, analytics, or any other form of data from its users.

Data Storage

This extension does not store any data locally or remotely. It does not use cookies, local storage, or any external databases.

Data Transmission

This extension does not transmit any data to external servers. All processing happens locally in your browser tab. The extension only reads the DOM content of the currently active Notion page tab to parse it into presentation slides.

Permissions

The extension requests the following permissions:

  • activeTab — To access the currently active tab when the user clicks the extension icon.
  • scripting — To inject the presentation overlay into the Notion page.
  • Host permissions (*.notion.site, *.notion.so) — To run the content script only on published Notion pages.

Third-Party Services

This extension does not use any third-party services, APIs, or SDKs.

Changes

Any changes to this privacy policy will be reflected on this page with an updated date.

Contact

For questions about this privacy policy, please open an issue on the extension's support page.
